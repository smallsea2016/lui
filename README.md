# lui

> LUI，L即light的缩写，是一套原生驱动的轻量级移动端UI组件库。


#### 原生能力
  ​ 基于html,css,js原生能力驱动，符合W3c标准规范
    
#### 专注UI
  LUI不是框架，她主要是为你的应用提供页面UI组件组织以及一些实用API，架构层面随你自行组织

#### 轻量易用
  核心css不超过35kb，未压缩的核心js未超过35kb。面向浏览器直接运行，不依赖babel，npm，nodejs，gulp，webpack等环境和各种工具
